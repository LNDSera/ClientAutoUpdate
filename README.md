## Системные требования
* Windows 10 64 OS
* Устанровленный [.NET Core 3.1 среда выполнения](https://dotnet.microsoft.com/download/dotnet-core/3.1)

## Приумущество 

*  Меньшее использование оперативной памяти
*  Простота в использовании (Надеюсь на это)
*  Стиль Win10

## Функции

- [x] Запуск игры
  - [x] Автоматическое обнаружение JRE пути
  - [x] Разделение версий
  - [x] 1.13+ Поддержка
  - [x] Forge Вариант
  - [x] Fabric Вариант
  - [x] Snapshot Вариант

- [x] Проверка авторизации
