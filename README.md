## Системные требования
* Windows 10 64 OS
* Установленный [.NET Core 3.1 среда выполнения](https://dotnet.microsoft.com/download/dotnet-core/3.1)

## Приумущество 

*  Меньшее использование оперативной памяти
*  Простота в использовании (Надеюсь на это)
*  Стиль Win10
